# 🧮 数学小工具

两个经典数学游戏的合集，含导航页统一切换。

## 📦 文件结构

```
D:\code\lxr\math\
├── index.html          导航首页
├── game24.html         极速 24 点
├── game24.css          24 点样式
├── game24.js           24 点逻辑
├── 99-table.html       大九九乘法表
└── 99-quiz.html        逐题闯关测验
```

## ⚡ 极速 24 点

经典 24 点卡牌游戏，用加减乘除和括号将 4 张扑克牌凑出 24。

- **三级难度**：初级 1-10 / 中级 1-13 / 高级经典困难组合
- **多解积分**：同一题可找多个不同解法，累积积分
- **三态主题**：夜间 / 日间 / 护眼模式循环切换
- **持久化**：积分、连胜、主题偏好保存于 localStorage

## 📐 大九九乘法表

11×1 ~ 19×9 共 81 条乘法口诀展示，带导出图片功能。

## 🎯 逐题闯关

大九九乘法表的交互测验。每次一题，答对自动进入下一题，积分不断累加。

- **81 道全随机**：每次打乱顺序
- **积分系统**：每题 +5 分，积分持久化保存
- **进度条**：实时显示闯关进度
- **完成庆祝**：全部通关后展示本局总积分
- **完美连击**：追踪连续答对次数

## 🚀 使用方式

直接用浏览器打开 `index.html`，点击卡片进入对应的应用。
所有数据保存在浏览器本地 localStorage 中。
